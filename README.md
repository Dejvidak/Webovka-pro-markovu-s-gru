# Aureum Vellum – návrh webu podle `web.csv`

Hotový je první statický návrh homepage + sekcí podle zadání.

## Co je v tomto MVP
- Elegantní bílo-zlatý styl, responzivní layout.
- Struktura menu včetně podsekcí Naši psi / Štěňata.
- Sekce: Hero, O nás, Naši psi, Štěňata, Zdraví a dokumentace, Novinky, Sociální sítě, Kontakt, budoucí E-shop a Akce.
- Kontaktní formulář s validací, GDPR checkboxem a honeypot antispam polem.
- Předvyplnění pole vrhu při kliknutí z karty aktuálního vrhu.
- Základní SEO metadata + `robots.txt` + `sitemap.xml`.

## Co je zjednodušené oproti plnému zadání
- Jde o statický frontend bez CMS, databáze a backend e-mail odeslání.
- Blog, detailní profily psů, vrhy a rezervace akcí jsou zatím jako obsahový základ / placeholder.
- Odkazy na IG/FB jsou ukázkové.

## Další doporučený krok
- Převést šablonu do CMS (např. WordPress + vlastní typy obsahu) tak, aby šlo editovat psy, vrhy, články, dokumenty a formuláře v administraci.
