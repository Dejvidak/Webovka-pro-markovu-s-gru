# Aureum Vellum – návrh webu podle `web.csv`

Aktualizované MVP po připomínkách.

## Co je hotové
- Navigace je nyní **rozklikávací** (multi-view), ne pouze skrolování po jedné dlouhé stránce.
- Dropdown menu „Naši psi“ a „Štěňata“ je funkční i na mobilu.
- Přidány **referenční fotografie** (hero + karty sekcí + karty psů/vrhů/novinek).
- Vylepšená grafika: jemné gradienty, lepší karty, stíny, modernější vizuál, čistší hierarchy.
- Zůstává kontaktní formulář s validací, GDPR checkboxem, honeypot antispam ochranou a potvrzením odeslání.

## Omezení MVP
- Stále jde o statický frontend bez CMS, databáze a backend odesílání e-mailů.
- Odkazy na IG/FB a část obsahu slouží jako referenční placeholder.

## Další doporučený krok
- Převést layout do CMS a napojit datové modely pro psy, vrhy, články a dokumentaci.
